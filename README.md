##  About the App  

PiggyPocket is a simple and user-friendly budgeting app designed to help users manage their finances effectively.  
It allows users to track income, monitor expenses, and maintain financial discipline.

## Installation & Setup  

Follow these steps to run the project using Android Studio:

1. Install Android Studio  
2. Open Android Studio  
3. Click **Open Project**  
4. Select the project folder  
5. Allow Gradle to sync  
6. Click ▶️ Run to start the app
